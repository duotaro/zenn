# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)



## 記事作成
> $ npx zenn new:article


## preview
> $ npx zenn preview 
localhost:8000　で閲覧可能。

> npx zenn preview --port 3000
ポート指定が可能

## git add / commit / push
> $ git add .
